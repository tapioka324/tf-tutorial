# tf-tutorial

## Environments
- Ubuntu 16.04 LTS
- CUDA 8.0 (cuDNN 6.0)
- Python 3.5.2
- tensorflow-gpu==1.4.1

## Usage
1. Download MNIST dataset
```
$ sh ./download.sh
```
1. Start
```
$ python3 mnist_softmax.py
```
